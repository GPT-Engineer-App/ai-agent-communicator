# ai-agent-communicator

A user interface that allows me to communicate with my personal AI Agent. Make the UI prompt the user for the ID of the AI agent. Take the Agent ID and plug it into the application using Open AI's Assistants API.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/ai-agent-communicator.git
cd ai-agent-communicator
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
